About SmartHomeGuy

Welcome!

I'm passionate about home automation, IoT devices, and making technology work smarter for everyday life. This blog is my digital workshop where I share projects, code snippets, tutorials, and lessons learned from automating my home.

What You'll Find Here

  • Home Automation Projects: Practical guides for Home Assistant, Node-RED, and other automation platforms
  • IoT Development: ESP32, Arduino, and Raspberry Pi projects with complete code
  • Code Snippets: Reusable Python scripts, YAML configurations, and automation blueprints
  • Product Reviews: Honest assessments of smart home devices and platforms
  • Troubleshooting: Solutions to common (and uncommon) smart home challenges

My Setup

My current smart home stack includes:

  • Home Assistant (Supervised on Raspberry Pi 4)
  • Node-RED for complex automations
  • ESPHome for custom sensors and devices
  • Zigbee2MQTT for device communication
  • Custom Python scripts for advanced automation
